# MySQL()

The MySQL entity provides three functionalities for interacting with MySQL databases: connecting, querying, and disconnecting. This functionality is provided by wrapping the mysql npm module. For additional information on how connections and queries should be made, see the documentation for mysql.

Parameters

Name Type Description
this.Par.ConnectionObject object

An object holding all the options for connecting to the MySQL database.

Returns

Void

# Setup(com)

The Setup command will set up a reference to the mysql npm module. The reference to mysql is saved to Vlt.Mysql.

# Start(com)

The Start command will set up a connection to the database server using the connection information provided in this.Par.ConnectionObject. A reference to the connection is saved in Vlt.Connection.

# Query(com)

The Query command recieves an SQL query (com.Sql), which may have placeholders, and a list of inserts to replace the placeholders. Then the SQL query is constructed and sent to the database connection, referenced at Vlt.Connection.

Parameters

Name Type Description
com object

The command object.
com.QueryObject object

An object containing the SQL query and any query options.

Returns

array

com.Results An array of result objects.

array

com.Fields An array with details about the table fields.

# Disconnect(com)
